写真やコメントを、当時の位置情報とともにマップに投稿できるウェブアプリを制作しました。
ユーザーごとに投稿できるため、誰が投稿したか分かったり、マップピンがユーザー独自の色になったりするようにしています。

マップのデータベースやユーザー情報はSQLフォルダをご参照ください。
